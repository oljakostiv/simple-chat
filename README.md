## Quick-start instructions:

In the **backend** directory, you can run:

### `npm run dev:lp` _- Long Pulling;_
### `npm run dev:es` _- Event Sourcing;_
### `npm run dev:ws` _- Web Socket;_

In the **client** directory:
### `npm start`
